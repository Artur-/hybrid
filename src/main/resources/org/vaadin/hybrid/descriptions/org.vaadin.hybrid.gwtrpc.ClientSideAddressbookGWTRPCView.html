Client-side GWT-RPC Example

This example has been completely implemented on the client-side.

It consists of the following classes:

  • AddressbookEditor: UI implementation compiled to JavaScript.
  • AddressbookRPCService: Public service interface for accessing the backend. This is implemented by the service servlet.
  • AddressbookRPCServiceImpl: Servlet implementing the service proxy to the backend.
  • AddressbookRPCServiceAsync: Asynchronous version of the service interface. This is used from the client to access the service over HTTP.
  • ClientSideAddressbookGWTRPCView: Server-side view that puts the widget on the page. In a GWT app this would be replaced by a static HTML page.
  • ClientSideAddressbookGWTRPCViewConnector: Initializer for the client-side widget. In a GWT app this would be replaced by the EntryPoint.
