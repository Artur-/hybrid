Client-side Connector Example

This example has been completely implemented on the client-side using Vaadin Connectors for communication with the backend.

It consists of the following classes:

  • AddressbookEditor: UI implementation compiled to JavaScript. Communicates with server-side AddressbookView.
  • AddressbookEditorServerRpc: RPC interface which AddressbookEditor uses to communicate with server-side AddressbookView (client to server).
  • AddressbookEditorClientRpc: RPC interface which server-side AddressbookView uses to communicate with AddressbookEditor (server to client).
  • AddressbookEditorState: Contains addresses which are automatically synchronized (server to client) automatically.
  • AddressbookView: Server-side view that puts the widget on the page. Handles RPC calls from AddressBookEditor and forwards to the backend.
  • AddressbookConnector: Initializer for the client-side widget.
